# Java + JavaFX Guess Who 

A single-player desktop version of the classic **Guess Who** game, built with Java and JavaFX.  
This project helped me strengthen my understanding of Java, GUI development, and event-driven logic.  
The full application is functional, interactive, and ready to play!

## Overview
This game recreates the classic Guess Who experience with turn-based questioning and strategic character elimination. The interface is built using JavaFX and SceneBuilder, offering a clean and intuitive user experience.

## Features
- **Turn-Based Gameplay**  
  Players have up to **5 questions** and **3 guesses** to identify the hidden character.
- **Interactive UI**  
  Clickable character cards allow players to visually eliminate options.
- **Real-Time Feedback**  
  The game updates dynamically as questions are answered and guesses are made.
- **Fully Functional Logic**  
  Includes question validation, elimination mechanics, and win/lose conditions.

## Tech Stack
- **Java**
- **JavaFX**
- **SceneBuilder**

## Skills Demonstrated
- Java  
- JavaFX GUI Development  
- Event-Driven Programming
